Copy selected text to clipboard (copy/paste buffer):

Command + C
Paste from buffer:

Command + V
Cut selected text to buffer (selected text is deleted):

Command + X
Save current file in VSCode:

Command + S
Toggle block comment in VSCode:

Option + Shift + A
Toggle line comment in VSCode:

Command + /
Move cursor from word to word right:

Option + Right arrow
Move cursor from word to word left:

Option + Left arrow
Select word right:

Option + Shift + Right arrow
Select word left:

Option + Shift + Left arrow
Select all contents:

Command + A
Select current line, including end of line character:

Command + Shift + Right arrow or Command + Shift + Left arrow
Jump to start of line:

Command + Left arrow
Jump to end of line:

Command + Right arrow
Jump to beginning of current file:

Command + Home
